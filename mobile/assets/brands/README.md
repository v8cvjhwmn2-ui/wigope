# Brand artwork

Bundled reward-deal tile artwork lives here. `WigopeOfferCard` renders these
with `tileAssetPath` so the complete square tile, discount cap, and logo match
the supplied reference design.

| Filename | Brand |
|---|---|
| `flipkart_tile.png` | Flipkart |
| `amazon_tile.png` | Amazon |
| `myntra_tile.png` | Myntra |
| `dominos_tile.png` | Domino's |
| `swiggy_tile.png` | Swiggy |
| `zomato_tile.png` | Zomato |
| `makemytrip_tile.png` | MakeMyTrip |
| `see_all_tile.png` | See All |

**Sizing**: square PNG, 192×192 or larger. If a future brand only has a logo
mark, use `assetPath` instead of `tileAssetPath`; the card can still fall back
to the painted brand-color tile.
